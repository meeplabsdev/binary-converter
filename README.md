# Binary Converter

Need to quickly convert a decimal or hex number (0-F) into binary? Now you can, with the press of two buttons.

## How to build
Just order the PCB & components, open up the KiCAD file and build it accordingly.

## BOM
- 2 	Battery holder
- 8	  NPN Transistor
- 8 	47k Resistor
- 4  	4.7k Resistor
- 4 	220 Resistor
- 1 	Potentiometer
- 8 	Push Button
- 4   LED
